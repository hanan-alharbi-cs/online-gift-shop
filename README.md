# Online Gift Shop

This project is a simple web-based e-commerce application that allows users to browse gift products, add items to a shopping cart, and manage their selections.

## Description

The system simulates an online gift store with basic shopping functionality.
Users can view products, add them to a cart, remove items, and see the total price dynamically.

## Features

* Product listing with images, names, and prices
* Add items to shopping cart
* Remove items from cart
* Cart data stored using LocalStorage
* Dynamic total price calculation

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

* `index.html` – Main webpage
* `style.css` – Styling and layout
* `script.js` – Application logic
* `images/` – Product images
